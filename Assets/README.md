
# Virtual Hell-Scape

## About
Video Hell-Scape is a two-player cooperative base defense game where players must protect a VHS tape by strategically rewinding and fast-forwarding objects in the environment to clear enemy waves.

## Logo & Poster
![Testing](POSTER2withlogo.png)

## Control
D-pad: Move
A button : Pick up & Put down modules
B button : Rotate
Hold A : Use Rewind skill
Hold B : Use Fast forward skill
